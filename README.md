##Question 1
Data is a collection of raw facts that have not been processed yet, but can be useful when analyzed and organised.
Examples:
1.Temperature readings of a city in 24 hours. 
2.The list of names in a class register.

##Question 2
Structured data is highly organised in a fixed format, like a table while unstructured data does not follow a fixed format or a table structure.
Examples of structured data:
1.An Excel spreadsheet showing student IDs and their corresponding grades.
2.A digital bank statement listing dates, descriptions, and transaction amounts.
Examples of unstructured data:
1.A collection of voice notes or audio recordings.
2.Messages sent through apps like Whatsapp.

##Question 3
A Schema is a blueprint or in simple terms,a "map" ,that defines how data is organised within a database.
Example:
Think of a contact list on a phone. The schema for a "Contact" might look like this:
 .Names:(Letters only)
 .Phone Number:(Numbers only)
 .Email Address(MUST contain an "@" symbol)
